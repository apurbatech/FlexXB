Classpublic final class FxBEngine
InheritanceFxBEngine Object

Entry point for AS3-? (de)serialization. Allows new annotation registration. The main access point consist of two methods: serialize() and deserialize, each corresponding to the specific stage in the conversion process. By default it registeres the built-in xml handling annotations through the xml description context.
