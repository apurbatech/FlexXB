Classpublic class SerializationEvent
InheritanceSerializationEvent flash.events.Event

Event being raised by the serialization mechanism in order to signal various stages teh process is in. The event is raised for each complex object/subobject being processed. There are four types of events one should listen for to hook into the mechanism:
  • preserialize - raised before the serialization of an object begins
  • postserialize - raised after the objet's serialization ends
  • predeserialize - raised before the deserialization of data to an object begins
  • postdeserialize - raised after the deserialization ends
