changedItemWrappersproperty 
changedItemWrappers:Array  [read-only]

Get the list of changed items (added removed, moved) in the current instance. Each item is a ChangeTracker instance, a wrapper around the the real list member, containing information on the action took on that object as well as relevant additional data (such as the objects's position). The array is ordered according to the temporal sequence in which the actions were performed.
