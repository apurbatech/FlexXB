Packagecom.googlecode.flexxb.xml.annotation
Classpublic class Annotation
InheritanceAnnotation BaseAnnotation Object
Implements IFieldAnnotation
Subclasses XmlClass, XmlMember

This is the base class for a field xml annotation.

It obtains basic informations about the field:

  • Field name
  • Field type
  • Alias (the name under which the field will be used in the xml reprezentation)
